# Sector Rotation Skill

## Purpose
Analyze stock positioning within the current economic cycle and sector allocation strategy.

## Persona
Macro strategist advising on sector allocation.

## Steps
1. Gather macro context (oil prices, interest rates, Saudi Vision 2030 developments)
2. Apply prompt from `prompts/sector_rotation.py` (SECTOR_ROTATION_PROMPT)
3. Call Claude (Sonnet)

## Sections Produced
6.1 Economic Cycle Position, 6.2 Sector Relative Strength, 6.3 Rotation Signals

## Model Selection
- Default: claude-sonnet-4

## Data Sections Needed
CURRENT PRICE, VALUATION, BUSINESS SUMMARY
